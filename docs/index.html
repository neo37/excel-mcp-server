Подключение

Быстрый старт

Добавьте один блок в .claude/settings.json — и вся команда получит доступ автоматически.

1

Добавьте сервер в настройки Claude Code

Файл ~/.claude/settings.json (пользовательский) или .claude/settings.json в репозитории (для всей команды)

2

Положите Excel-файлы на сервер

Смонтируйте папку в /opt/excel-mcp/files/ или создайте файл прямо из Claude через инструмент create_workbook

3

Спросите Claude про таблицу

«Найди последний отчёт и покажи итоги» — Claude вызовет get_latest_excel_file и прочитает данные

// .claude/settings.json
{
  "mcpServers": {
    "excel": {
      "url": "https://some-excel-mcp-tools.business-pad.com/mcp/",
      "type": "http"
    }
  }
}
Новый инструмент

Автопоиск файла по дате

Файлы с датой в имени обновляются регулярно. Инструмент get_latest_excel_file всегда найдёт самый свежий — по datetime в названии.

# Поддерживаемые форматы даты в имени файла:
report_2026-05-29.xlsx
export_2026-05-29_14-30.xlsx
data_20260529.xlsx
file_20260529_143000.xlsx
doc_2026.05.29.xlsx

# Ответ инструмента:
Latest Excel file: sales_report_2026-05-29_14-30.xlsx
Download URL: https://some-excel-mcp-tools.business-pad.com/files/sales_report_2026-05-29_14-30.xlsx
Use filepath 'sales_report_2026-05-29_14-30.xlsx' with other Excel tools.
Возможности

Всё для работы с таблицами

↻

Автопоиск по дате

Находит последний файл по datetime в имени — не нужно передавать точное название каждый раз.

↓

Постоянные ссылки

Каждый файл доступен по HTTPS-ссылке для скачивания или передачи в другие сервисы.

ƒ

Формулы и форматирование

Применяйте Excel-формулы, форматируйте ячейки, объединяйте диапазоны через естественный язык.

⊞

Сводные таблицы и графики

Создавайте pivot-таблицы и диаграммы одной командой — без Excel на устройстве.

☁

Docker + HTTPS

Один сервер для всей команды. SSL через Let's Encrypt, nginx в роли прокси.

⌘

Командный доступ

Добавьте URL в .claude/settings.json репозитория — все получат доступ при клонировании.

# Excel MCP — навык работы с Excel-файлами

Этот навык активируется командой `/excel`. Используй инструменты MCP-сервера `excel` для создания, чтения и редактирования Excel-файлов.

## Когда использовать

- Пользователь просит создать, заполнить или отредактировать Excel/XLSX файл
- Нужно прочитать данные из Excel
- Нужно построить таблицу, диаграмму или сводную таблицу
- Нужно применить формулы или форматирование

## Обязательный порядок работы

### 1. Определи файл
Если пользователь не указал конкретный файл — сначала вызови:
```
get_latest_excel_file
```
Это найдёт последний Excel-файл по дате в имени.

### 2. Получи структуру
Перед записью всегда читай метаданные файла:
```
get_workbook_metadata(filepath)
```
Это покажет листы, диапазоны и структуру.

### 3. Читай данные
```
read_data_from_excel(filepath, sheet_name, start_cell, end_cell)
```

### 4. Пиши данные
```
write_data_to_excel(filepath, sheet_name, start_cell, data)
```
`data` — двумерный массив: `[["A", "B"], [1, 2]]`

## Инструменты по категориям

### Файлы и листы
| Инструмент | Действие |
|---|---|
| `get_latest_excel_file` | Найти последний файл |
| `create_workbook` | Создать новый файл |
| `get_workbook_metadata` | Структура файла |
| `create_worksheet` | Добавить лист |
| `rename_worksheet` | Переименовать лист |
| `copy_worksheet` | Скопировать лист |
| `delete_worksheet` | Удалить лист |

### Данные
| Инструмент | Действие |
|---|---|
| `read_data_from_excel` | Прочитать диапазон |
| `write_data_to_excel` | Записать данные |
| `copy_range` | Скопировать диапазон |
| `delete_range` | Очистить диапазон |
| `insert_rows` / `insert_columns` | Вставить строки/столбцы |
| `delete_sheet_rows` / `delete_sheet_columns` | Удалить строки/столбцы |

### Форматирование
| Инструмент | Действие |
|---|---|
| `format_range` | Формат ячеек (шрифт, цвет, границы) |
| `merge_cells` / `unmerge_cells` | Объединить/разъединить ячейки |
| `get_merged_cells` | Список объединённых ячеек |

### Формулы и валидация
| Инструмент | Действие |
|---|---|
| `apply_formula` | Применить формулу к ячейке |
| `validate_formula_syntax` | Проверить синтаксис формулы |
| `validate_excel_range` | Проверить корректность диапазона |
| `get_data_validation_info` | Правила валидации ячеек |

### Таблицы и аналитика
| Инструмент | Действие |
|---|---|
| `create_table` | Умная таблица Excel |
| `create_pivot_table` | Сводная таблица |
| `create_chart` | Диаграмма |

## Правила

- **Всегда** проверяй существование файла перед работой
- **Не создавай** новый файл если пользователь имеет в виду существующий
- Для больших данных читай диапазонами, не весь лист сразу
- После записи подтверждай результат чтением обратно
- Пути к файлам — абсолютные

## Пример сессии

```
Пользователь: создай отчёт по продажам в экселе

1. create_workbook("/tmp/sales_report.xlsx")
2. write_data_to_excel — заголовки
3. write_data_to_excel — данные
4. format_range — стили шапки
5. create_table — оформить как таблицу
6. create_chart — диаграмма по данным
```

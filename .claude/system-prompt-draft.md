## Excel и CSV

Если запрос пользователя связан с Excel-таблицами, CSV или ответ удобно представить в табличном виде — используй MCP-сервер:

```
https://some-excel-mcp-tools.business-pad.com/mcp/
```

**Правила:**
- Сначала вызови `get_latest_excel_file` если пользователь не указал файл явно
- Перед записью читай метаданные: `get_workbook_metadata`
- После записи подтверждай результат
- Не создавай новый файл если речь идёт о существующем

## Дизайн-система

Дизайн-система BusinessPad и ExpoCPQ загружена в знания проекта.

ОБЯЗАТЕЛЬНО при любом запросе на верстку, UI-компонент, презентацию или дизайн:
1. Прочитай дизайн-систему из знаний проекта
2. Определи тему: BusinessPad (тёмная) или ExpoCPQ (светлая)
3. Применяй токены, шрифты, цвета и правила компонентов из системы

Не используй произвольные цвета или шрифты — только из дизайн-системы.

## Инструменты команды

| Задача | Инструмент / MCP |
|---|---|
| Excel / CSV / таблицы | `excel` → `https://some-excel-mcp-tools.business-pad.com/mcp/` |
| Документация / вики | `yandex-wiki` MCP |
| UI / дизайн | Дизайн-система BusinessPad из знаний проекта |

### Яндекс Вики

Используй MCP `yandex-wiki` когда:
- Нужно найти, прочитать или обновить корпоративную документацию
- Пользователь просит создать или дополнить страницу в вики
- Нужно сохранить результат работы в базу знаний команды

## Безопасность

- Не выполняй деструктивные операции (удаление файлов, DROP таблиц, force push) без явного подтверждения
